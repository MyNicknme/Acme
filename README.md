🚀 Однокнопочный скрипт для получения ACME-сертификата
(для локального IP VPS)

🔹 Быстрый запуск
bash <(curl -Ls https://raw.githubusercontent.com/MyNicknme/Acme/refs/heads/main/Acme-yonggekkk.sh)

📌 Функции и особенности
Поддержка VPS:
только IPv4
только IPv6
Dual Stack (IPv4 + IPv6)
Поддержка двух режимов выпуска сертификата:
через порт 80 (HTTP-01)
через DNS API
Поддерживаются как обычные домены, так и wildcard (подстановочные *.domain.com).
Поддержка DNS API у провайдеров:
Cloudflare
Tencent DNSPod
Alibaba Aliyun
Возможность:
просматривать текущие сертификаты
отзывать сертификаты
удалять уже выпущенные сертификаты
Ручное продление:
одним нажатием
или продление конкретного домена
Файлы сертификатов сохраняются в:
/root/ygkkkca
Интеграция со следующими прокси-скриптами
(можно использовать один общий сертификат):
4protocol
x-ui
⚠️ Важно
При использовании режима через порт 80:
порт 80 будет принудительно освобождён
не рекомендуется одновременно использовать nginx, caddy или другие сервисы с авто-получением сертификатов
